# Simple-Web-App

   Web application connects to a database on kubernetes cluster, reads and returns those data's upon HTTP request.
     
   
   # Prerequisites
      
     - Docker 1.13.1
     - Helm 3.4.2
     - MiniKube 1.15.1
     - Kubectl 1.19
     - PV provisioner support in the underlying infrastructure
